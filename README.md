**Alargon's Lua Obfuscator & Deobfuscator (Hex-Based)**

A simple yet effective Lua code obfuscation tool that converts your source code into hexadecimal representation with a built-in loader. Perfect for protecting your Lua scripts!

**Features:**
- 🔒 Converts Lua code to HEX format
- 📦 Automatic script loader generation
- 📁 Auto-saves obfuscated files to `/obfuscated` folder and deobfuscated files to `/deobfuscated` folder
- 🔄 Built-in deobfuscator to restore original code
- 🎯 Unique filename generation (no overwrites)
- 🚀 Task.spawn() based execution for modern Lua environments

**How it works:**
1. Paste your Lua code
2. Get obfuscated script with HEX-encoded content
3. The obfuscated script self-decodes and executes using task.spawn()
4. Use deobfuscator to restore original code when needed

**Perfect for:**
- Protecting game scripts (Roblox, FiveM, etc.)
- Hiding source code logic
- Quick and simple code obfuscation
- Learning about basic obfuscation techniques

- Run `obfuscator.py` to obfuscate your Lua code
- Run `deobfuscator.py` to restore obfuscated scripts
- Output files are automatically saved in respective folders
